# EMOTE – Emotion Detection using BiLSTM

This project implements an emotion detection system using a Bidirectional LSTM (BiLSTM) neural network.

## Overview
The EMOTE system classifies emotions from input text by learning contextual information using deep learning.

## Dataset
- GoEmotions dataset (Google)
- Selected emotions: joy, sadness, anger, fear

## Methodology
1. Text preprocessing and cleaning
2. Tokenization and padding
3. Word embedding layer
4. BiLSTM neural network
5. Class-weighted training
6. Emotion prediction for new text

## Results
The trained model correctly predicts emotions such as joy and sadness from input sentences.

## How to Run
1. Open `SCT.ipynb` in Google Colab
2. Upload the dataset `goemotions_1.csv`
3. Run all cells sequentially
